Skema Rangkaian & Langkah Kerja

Skema sederhana: Sensor → Mikrokontroler → Driver → Servo

Contoh langkah kerja singkat:

  1. Pasang sensor pada sisi tempat sampah untuk mendeteksi jarak
  2. Hubungkan sensor ke Arduino (Vcc, GND, Trig/Echo atau output digital)
  3. Program Arduino untuk membaca sensor dan menggerakkan servo
  4. Pasang mekanik tutup pada servo
  5. Uji coba dan kalibrasi jarak sensor

Berikut contoh potongan kode Arduino (contoh singkat):

// Contoh: sensor ultrasonik + servo
#include <Servo.h>
const int trigPin = 9;
const int echoPin = 8;
Servo tutup;
void setup(){
  tutup.attach(3);
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
  Serial.begin(9600);
}
long readDistance(){
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);
  long dur = pulseIn(echoPin, HIGH);
  return dur * 0.034 / 2; // cm
}
void loop(){
  long d = readDistance();
  Serial.println(d);
  if(d < 25){ // jika jarak kurang dari 25 cm
    tutup.write(90); // buka
  } else {
    tutup.write(0); // tutup
  }
  delay(200);
}
